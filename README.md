## 为什么用 FunCV

复制之后经常要：找历史、格式化 JSON、压一行 SQL、看时间戳、查个英文单词……  
FunCV 在本地完成监听、存储与推理，**默认不上传云端**，并通过插件协议支持 Node / Python / Go / Shell 扩展。  
右侧内容区会**自动识别语言并语法高亮**，推荐解析一键应用，编辑草稿不影响历史原文。

## 功能一览

### 剪贴板历史

- **系统剪贴板监听**：轮询整板内容（非键盘 hook），支持 `Ctrl/Cmd+C`、右键复制等
- **文本 + 图片**：文本入库；图片存 PNG，可预览并写回剪贴板
- **图片文字识别（本地 OCR）**：图片详情点「识别文字」
  - macOS：系统 **Vision**（无需联网/下模型）
  - Windows：系统 **Windows.Media.Ocr**（内置插件「图片文字识别」，可在插件列表开关）
- **历史管理**：搜索、选中预览、双击复制、删除、清空
- **钉住 / 收藏**：内容区图标；历史列表中保留标记且不消失；**钉住 / 收藏** Tab 用于快速筛选；且不易被最大条数清理  
- **收藏备注**：收藏后可写备注，顶部搜索会匹配备注内容
- **hash 去重**：相同内容合并更新，避免刷屏
- **搜索框计算器**：输入 `1+1`、`(2+3)*4`、`sqrt(9)` 等公式时自动识别，结果展示在右侧**内容区**，可一键复制（仍同时搜索历史）
- **搜索触发插件**：在搜索框输入触发词，匹配已安装插件后在内容区出结果；内置「时间」等**默认不安装**，可在插件面板一键安装

### 内容预览 · 语法高亮

右侧内容区对文本提供**可编辑草稿 + 语法高亮**（编辑只改面板草稿，**不改写历史记录**）：

| 能力 | 说明 |
| --- | --- |
| 语言识别 | 自动识别 **JSON / SQL / Go / JavaScript / TypeScript / Python** 等 |
| 语法高亮 | 关键字、字符串、数字等分色显示；标题栏显示语言芯片（如 `SQL`、`JSON`） |
| 可编辑草稿 | 面板内直接改字；复制时可带出当前面板内容；「恢复原文」一键回退 |
| 界面 | 内容区无滚动条干扰，仍可滚轮 / 触控板滚动 |

解析建议（如 SQL 格式化、JSON 美化）应用后，会按规则提示同步高亮语言。

### 智能推荐解析

右侧**默认显示原文**；底部「推荐解析」展示规则与预览，**单击应用 / 双击复制**：

| 类型 | 能力 |
| --- | --- |
| JSON | 单行格式化 / 多行压缩 / 清洗转义后格式化 |
| SQL | 格式化（关键字大写 + 缩进）/ 压缩为一行；支持带 `\n` 等转义的粘贴 SQL 清洗后格式化 |
| 时间戳 | Unix 秒/毫秒 ↔ 可读时间 |
| URL / UUID / 颜色 / Base64 | 识别与规范化 |
| 插件结果 | 启用插件后自动参与推荐 |

### 通用插件系统

- 顶部 **插件** 面板，**列表 / 自定义** 双 Tab
- **列表**：统一插件卡片（含可选安装的搜索触发类、剪贴板内容类）、开关 / 安装卸载、上传删除；ECDICT 词典管理
- **自定义**：完整接口规范 + 下载 Node / Python / Go / Shell 示例
- 内置可选「时间」：默认不安装；安装后搜索框输入 `时间` / `time` / `now` 等 → 内容区显示 Unix 秒/毫秒与多时区时间
- **统一协议**（仅 2 个入参）：

```json
{ "content": "剪贴板内容", "type": "text" }
```

`type` 为 `"text"` | `"img"`（可扩展）。插件从 **stdin** 读 JSON，向 **stdout** 写结果。

### 内置 · 英汉互译

- 数据源：[skywind3000/ECDICT](https://github.com/skywind3000/ECDICT)
- 本地下载导入 SQLite，**离线查词**
- 英文 → 中文释义；中文 → 反查英文词条
- **半屏内联推荐**：原文/图片不超过内容区约一半高度时，**下半区**直接展示推荐解析卡片（单击应用、双击复制；时间戳/词典为点按复制）；内容较长时仍用底部推荐条

### 趣味统计

- 运行时记录每次剪贴板变更（文本 / 图片）
- 识别**重复复制**、**连点复制**
- 累计 / 今日次数、重复率、高峰时段、连续活跃天、Top 重复内容

### 桌面体验

- **无边框**主窗口，关闭进托盘（可隐藏 Dock）
- **全局快捷键**唤起 / 隐藏；未置顶时按快捷键会**置顶**而非误关
- **追加复制快捷键**（可配置 3 键组合，默认 `Cmd+Shift+C`）：剪贴板已是文本时，将选中内容用双空格追加到末尾；连续追加相同内容只生效一次
- **复制并解析快捷键**（可配置 3 键组合，默认 `Shift+Option+X` / `Shift+Alt+X`）：选中文本后一键复制并弹出 FunCV 查看解析，无需先手动复制再打开
- 设置：历史条数、轮询间隔、**监听类型（文本 / 图片，默认都开）**、唤起 / 追加复制 / 复制并解析快捷键  
- **数据导出 / 导入**：收藏与钉住（JSON+图片，导入可合并）/ 全部（复制或覆盖 `history.db`+`images`，便于迁移）

## 存储

**SQLite（WAL）** 统一承载历史与统计，Mac / Windows 同一套实现。

```text
~/Library/Application Support/NfunCv/   # macOS
%APPDATA%\NfunCv\                       # Windows
├── config.json
├── history.db              # SQLite + WAL（长连接）
│   ├── history             # 剪贴板历史
│   ├── clipboard_events    # 每次复制事件流水
│   ├── stats_global        # 累计计数
│   └── stats_daily         # 按日聚合
├── images/<id>.png
└── plugins/
    ├── registry.json
    ├── user/<plugin-id>/
    └── data/ecdict/
```

应用内 **统计** 面板可看：累计/今日复制、重复率、文本 vs 图片、连续活跃天、重复 Top、近 14 天趋势。

全部默认本地；词典下载为可选联网行为。

```bash
cd nfun-cv
npm install
npm run tauri dev
```

构建：

```bash
npm run build
npm run tauri build
```

### 使用提示

1. 任意复制文本或图片 → 左侧出现历史  
2. 点选记录 → 右侧看原文（代码/结构化文本自动高亮）；底部点推荐解析  
3. 可在右侧直接改草稿后复制；点「恢复原文」回到入库内容（历史本身不变）  
4. 顶部搜索框可当计算器：输入 `1+1` 等公式 → 右侧内容区显示结果并可复制  
5. **插件 → 列表** 安装「时间」→ 搜索框输入 `时间` / `now` 看时间戳与多时区  
6. **插件** → 下载 ECDICT → 复制英文单词即可英汉互译  
7. **插件 → 自定义** 下载示例 → 改脚本 → **列表 → 上传**  
8. 关闭窗口仍由托盘 / 快捷键控制

## 插件协议（摘要）

| 方向 | 格式 |
| --- | --- |
| 入参 | `{ "content": string, "type": "text" \| "img" }` |
| 出参 | `{ "ok": true, "title", "body", "preview", "hint" }` |
| 失败 | `{ "ok": false, "error": "..." }` |
| 清单 | `plugin.json`：`id` / `name` / `runtime` / `entry` / `types` |

运行时：`node` · `python` · `go` · `shell` · `builtin`

示例目录：[`plugin-examples/`](./plugin-examples/)
